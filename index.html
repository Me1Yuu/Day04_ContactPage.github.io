Day04 of ChatGPT Challenge

  1. Create a folder:
    Name it "day04-contact-page

  2. Inside the folder, create:
    • "index.html" for your code

  3. Structure your HTML page with the following elements:
    • A <title> in the <head>
    • A <h1> for the page heading (e.g. "Contact Me")
    • A paragraph <p> expaining how someone can reach you or why they should
    • A form using the <form> tag
    • Inside the form, add:
      • An <input> for a name
      • An <input> for an email adress
      • A <texarea> for a message
      • A <button> to "send" the message (it won't actually send anything yet)
